   # 在window环境下运行py程序

 一种简单的方法是用批量处理命令打开.py文件，方法如下：

 1、在py文件同目录下新建一个txt文件，粘贴虾下面命令

 
```
 start py -3 "要运行的文件.py"
```
 2、修改文件后缀为bat文件，然后运行即可
